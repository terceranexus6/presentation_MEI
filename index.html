Definición

Conjunto de componentes de un circuito integrado que administra el flujo de datos entre el procesador, la memoria y los periféricos.

En un PC, distinguimos que tiene principalmente dos partes: northbridge y southbridge.

El northbridge se conecta directamente al procesador via FSB (Front side bus). El northbridge tiene un controlador de memoria para acceso rápido.

El southbridge es más lento y la información que lo atraviesa debe pasar antes por el northbridge. Está conectado a otros periféricos como USB, SATA ò disco duro.
